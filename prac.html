⚙️ 核心逻辑 · 随机点名机制
🧠 实现原理： 利用 JavaScript 的 setInterval 定时器，每 70毫秒 从学生数组中随机抽取一名学生，动态更新页面中央的名字并同步高亮下方花名册对应项。
🎯 开始按钮： 清除已有定时器 → 立即随机抽取一次 → 启动定时器持续滚动（高频刷新）。
🛑 结束按钮： 清除定时器 → 滚动停止 → 当前屏幕显示的名字即为最终点中的学生，高亮固定。
📊 随机算法： Math.random() 生成随机索引，保证每次抽取独立且等概率。开始/结束可无限次操作，每次停止即产生一次公平点名结果。
✅ 额外设计：花名册实时高亮 + 流畅动画，点名过程直观清晰。
